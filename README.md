# NoteNote 🎵

**NoteNote** is a high-precision chromatic audio transcription tool that provides real-time visual feedback in the form of sheet music. It allows users to capture audio input and see it transcribed instantly using advanced frequency detection and visualization techniques.

## ✨ Features

- **Chromatic Note Detection**: Accurately detects and transcribes all 12 chromatic notes, including accidentals.
- **Real-time Sheet Music**: Visualizes captured notes on a musical staff using VexFlow.
- **120 BPM Metronome**: built-in rhythmic guidance for consistent transcription.
- **Visual Feedback**: Real-time highlighting of notes during playback.
- **Modern UI**: Clean, responsive design built with Vite and Vanilla CSS.

## 🚀 Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (latest LTS recommended)
- [npm](https://www.npmjs.com/)

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/YuHyungmin1226/notenote.git
   cd notenote
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

## 🛠️ Built With

- **Vite** - Next Generation Frontend Tooling
- **VexFlow** - Music notation rendering engine
- **Vanilla JavaScript** - Core logic and audio processing
- **Vanilla CSS** - Custom styling and animations
